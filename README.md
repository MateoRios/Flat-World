# Flat-World
Proyecto personal donde se desarrolla numerosos paisajes basados en Flat Design o diseño plano utilizando tan solo lenguajes front-end.
Se puede navegar a través de ellos mendian el menú lateral. Las paisajes estan diseñados mediante Adobe Photoshop y Adobe Illustrator,
y los efectos y animación de las escenas se realizan con CSS y JS.

Con este proyecto se busca mejorar los conocimientos sobre lenguajes front-end y sobre Flat Design. También aprender más sobre
programas de diseño como son Photoshop y Illustrator.
